[Nslookup command is a network administration command-line tool for querying the Domain Name System (DNS) to obtain domain name or IP address mapping, DNS records, and other DNS-related information. It is useful for troubleshooting network issues and verifying DNS configurations.]

## Nslookup Syntax:
```console
nslookup [options] [hostname]
```

## Options:
| Option        | Description                            |
|---------------|----------------------------------------|
| -querytype    | Specify the query type to send.        |
| -timeout      | Set the query timeout in seconds.      |
| -debug        | Enable debug mode.                     |

## Parameters:
| Parameter     | Description                            |
|---------------|----------------------------------------|
| hostname      | Specifies the domain name or IP address to query.  |

:::caution
Exercise caution while using nslookup as incorrect usage may lead to misconfiguration or unintended results.
:::

## Examples:

### Basic query
```console
nslookup example.com
```
Query the IP address of example.com.

### Reverse lookup
```console
nslookup 93.184.216.34
```
Perform a reverse lookup for the IP address.

### Specify query type
```console
nslookup -querytype=mx example.com
```
Query the Mail Exchange (MX) records for the domain.

### Debugging
```console
nslookup -debug example.com
```
Run nslookup in debug mode for troubleshooting.

### Timeout setting
```console
nslookup -timeout=5 example.com
```
Set a query timeout of 5 seconds.

:::

## FAQ:

### What is the syntax for a basic nslookup query?
To perform a basic nslookup query, use the following command:
```console
nslookup example.com
```

### How can I specify a query type in nslookup?
To specify a query type in nslookup, use the `-querytype` option followed by the desired query type. For example:
```console
nslookup -querytype=mx example.com
```

### How do I perform a reverse lookup using nslookup?
To perform a reverse lookup using nslookup, provide the IP address as the parameter. For example:
```console
nslookup 93.184.216.34
```

### How can I enable debug mode in nslookup?
To enable debug mode in nslookup, use the `-debug` option. For example:
```console
nslookup -debug example.com
```

### How do I set a timeout for nslookup queries?
To set a timeout for nslookup queries, use the `-timeout` option followed by the timeout value in seconds. For example:
```console
nslookup -timeout=5 example.com
```
